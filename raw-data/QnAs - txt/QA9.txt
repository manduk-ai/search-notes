Q: How to generate random string of letter and digits and of length N?

Q: How to open a file and read it line by line?

A: lines = file.readlines()

A:     print("Line {}: {}".format(index, line.strip()))

Q: Difference between normal and binomial distributions?
A: Binomial distribution is discrete and normal distribution is continuous.
A: In other words, there are a finite amount of events in a binomial distribution, 
A: but an infinite number in a normal distribution.

Q: Difference: GET vs POST vs PUT
A: GET is used to request data from a specified resource.
A: Query string is sent in the URL of a GET request.
A: POST/PUT are used to send data to a server to create/update a resource.
A: The data sent to the server is stored in the request body of the HTTP request
A: The difference between POST and PUT is that PUT requests are idempotent. 
A: That is, calling the same PUT request multiple times will always produce the same result. 
A: Calling a POST request repeatedly have side effects of creating the same resource multiple times.
